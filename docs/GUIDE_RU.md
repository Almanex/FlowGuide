## Введение

**FlowGuide** помогает создавать интерактивные пошаговые инструкции для любых Windows-приложений. Ваши гайды будут проводить пользователей через сложные процессы, подсвечивая нужные кнопки и поля.

### Что вы сможете делать:
* Записывать действия пользователя в любом приложении  
* Автоматически создавать текстовые инструкции  
* Редактировать и улучшать записанные шаги  
* Делиться готовыми гайдами с коллегами  

### Что вам понадобится:
* Windows 10 или 11
* Права администратора (для первого запуска)
* Приложение, для которого вы создаёте инструкцию

## Подготовка к записи

### Шаг 1: Запустите FlowGuide Recorder

1. Откройте папку `Release\Recorder\`
2. Запустите **FlowGuide.Recorder.exe** от имени администратора
   * ПКМ на файле -> "Запуск от имени администратора"
   * Или: один раз запустить обычным способом и разрешить права

> [Совет]: Создайте ярлык на рабочем столе для быстрого доступа

### Шаг 2: Подготовьте приложение

**Перед началом записи:**

1. Откройте приложение, для которого создаёте гайд (Word, 1С, и т.д.)
2. Установите начальное состояние (например, откройте нужное окно)
3. Закройте лишние окна и уведомления
4. Отключите автосохранение и авто-обновления (если возможно)

> [Внимание]: Рекордер будет записывать ВСЕ ваши клики, поэтому подготовьте рабочее пространство заранее!

## Создание нового гайда

### Этап 1: Настройка гайда

1. В главном окне FlowGuide Recorder заполните поля:

   **Название гайда** (обязательно)
   ```
   Пример: "Создание нового документа в Word"
   ```
   
   **Автор** (обязательно, по умолчанию - ваше имя пользователя)
   ```
   Example: "Иван Петров"
   ```
   
   **Описание** (опционально)
   ```
   Пример: "Пошаговая инструкция по созданию нового документа 
   с шаблоном и сохранением в нужную папку"
   ```

2. Нажмите кнопку **"Записать новый гайд"**

### Этап 2: Начало записи

**После нажатия кнопки:**

1. Главное окно Recorder свернётся
2. Появится **плавающая панель управления** (Control Panel) с кнопками:
   * **[Start Recording]** - начать запись
   * **[Pause]** - пауза
   * **[Stop]** - остановить запись

3. Нажмите **"Start Recording"**

**Что произойдёт:**
* Появится **красная рамка** вокруг элементов под курсором мыши
* Это означает, что запись началась
* Панель управления останется видимой - вы можете её перетащить в удобное место

> [Совет]: Расположите панель управления так, чтобы она не мешала, но была под рукой

### Этап 3: Запись действий

**Теперь выполняйте действия, которые хотите записать:**

#### Пример: Создание документа в Word

1. **Наведите курсор** на вкладку "Файл"
   * Вы увидите красную рамку вокруг вкладки
   
2. **Кликните** на "Файл"
   * Шаг записан!
   * Система автоматически создаст инструкцию: "Нажмите вкладку 'Файл'"
   * Сделает скриншот кнопки

3. **Кликните** на "Создать"
   * Ещё один шаг записан!
   * Инструкция: "Нажмите кнопку 'Создать'"

4. Продолжайте выполнять нужные действия...

**Важные моменты:**

* Кликайте точно на элементы - система запомнит эти кнопки/поля
* Делайте паузы между кликами (1-2 секунды) - так система успеет сделать скриншот
* Не торопитесь - лучше записать медленно и правильно
* НЕ кликайте на пустые области - это создаст лишние шаги

### Этап 4: Использование паузы

**Если нужно сделать перерыв:**

1. Нажмите **[Pause]** на панели управления
2. Запись приостановится
3. Вы можете:
   * Подготовить следующий этап
   * Проверить уже записанные шаги
   * Отдохнуть
4. Нажмите **"Start Recording"** для продолжения

> [Совет] Когда использовать паузу:
> * Нужно заполнить форму вручную
> * Ожидаете загрузку данных
> * Хотите проверить, что записалось

### Этап 5: Остановка записи

**Когда закончили записывать:**

1. Нажмите **[Stop]** на панели управления
2. Панель закроется
3. Автоматически откроется **Редактор шагов** (Step Editor)

## Редактирование гайда

### Окно редактора шагов

После остановки записи вы увидите список всех записанных шагов:

```
Step Editor - 8 steps recorded

[1] Нажмите вкладку "Файл"
Type: LeftClick
[Edit] [Delete]

[2] Нажмите кнопку "Создать"
Type: LeftClick
[Edit] [Delete]

[3] Введите текст в поле "Название"
Type: LeftClick
[Edit] [Delete]

...
```

### Редактирование отдельного шага

**Для редактирования шага:**

1. Нажмите кнопку **[Edit]** рядом с нужным шагом
2. Откроется окно редактирования

**Что можно изменить:**

#### 1. Текст инструкции
```
Было: "Нажмите кнопку 'Создать'"

Можно изменить на:
"Нажмите кнопку 'Создать', чтобы создать новый документ
из шаблона"
```

> [Совет]: Делайте инструкции понятными и подробными!

#### 2. Тип действия

**Normal Click** (обычный клик)
* Используется для большинства кликов
* Пользователь должен сам кликнуть на элемент

**System Action** (системное действие)
* Используется для системных элементов Windows
* Можно выполнить автоматически

**Когда использовать System Action:**

| Действие | Выбрать тип |
|---|---|
| Открыть Пуск | `OpenStartMenu` |
| Открыть Параметры Windows | `OpenSettings` |
| Открыть Проводник | `OpenFileExplorer` |
| Открыть Диспетчер задач | `OpenTaskManager` |

#### 3. MS-Settings команда (для системных действий)

Если выбрали **System Action: OpenSettings**, можно указать конкретный раздел:

```
ms-settings:display          - Настройки экрана
ms-settings:network          - Сеть и интернет
ms-settings:privacy          - Конфиденциальность
ms-settings:windowsupdate    - Центр обновления
```

#### 4. Показать кнопку автовыполнения

* Включено (по умолчанию) - пользователь увидит кнопку "Выполнить автоматически"  
* Выключено - пользователь должен сам выполнить действие

### Удаление шагов

**Если записали лишний шаг:**

1. Найдите его в списке
2. Нажмите кнопку **[Delete]**
3. Подтвердите удаление

> [Внимание]: Удаление шага нельзя отменить! Будьте осторожны.

### Изменение порядка шагов

**К сожалению, сейчас нельзя изменить порядок шагов через интерфейс.**

**Обходное решение:**
1. Остановите без сохранения
2. Перезапишите гайд в правильном порядке

> [Совет]: Планируйте последовательность действий заранее!

### Сохранение гайда

**Когда закончили редактирование:**

1. Нажмите кнопку **"Save Guide"** (зелёная, внизу справа)
2. Гайд сохранится в:
   ```
   C:\Users\ВашеИмя\Documents\FlowGuide\НазваниеГайда\
   ```
3. Увидите сообщение: "Гайд успешно сохранён!"

**Что сохраняется:**
* `guide.json` - описание всех шагов
* `step_1.png`, `step_2.png`, ... - скриншоты элементов

### Отмена сохранения

**Если передумали:**

1. Нажмите кнопку **"Cancel"** (серая)
2. Гайд НЕ будет сохранён
3. Все записанные шаги будут потеряны

## Редактирование существующего гайда

### Загрузка гайда для редактирования

1. В главном окне Recorder нажмите **"Редактировать существующий гайд"**
2. Выберите файл **guide.json** в папке нужного гайда:
   ```
   C:\Users\ВашеИмя\Documents\FlowGuide\НазваниеГайда\guide.json
   ```
3. Откроется редактор шагов с уже записанными шагами

### Что можно делать

* Редактировать текст инструкций  
* Изменять тип действия  
* Удалять ненужные шаги  
* Исключено: Добавлять новые шаги (нужно перезаписать гайд)  

### Сохранение изменений

1. Нажмите **"Save Guide"**
2. Изменения перезапишут старый файл
3. Старая версия будет потеряна

> [Совет]: Сделайте резервную копию папки гайда перед редактированием!

## Советы и лучшие практики

### Как создать качественный гайд

#### 1. Планирование

**Перед записью составьте план:**

```
Пример плана для "Создание документа в Word":

1. Открыть меню Файл
2. Выбрать Создать
3. Выбрать шаблон "Официальное письмо"
4. Заполнить поле "Получатель"
5. Заполнить поле "Тема"
6. Сохранить документ
```

* Запишите план на бумаге  
* Проверьте, что все шаги логичны  
* Убедитесь, что ничего не пропустили  

#### 2. Подготовка рабочего пространства

**Создайте идеальные условия для записи:**

* Закройте все ненужные окна  
* Очистите рабочий стол от лишних файлов  
* Отключите уведомления (режим "Не беспокоить")  
* Подготовьте тестовые данные заранее  
* Убедитесь, что приложение работает стабильно  

#### 3. Запись

**Правила записи:**

1. **Медленно и точно**
   * Делайте паузу 2-3 секунды между кликами
   * Система должна успеть сделать скриншот

2. **Кликайте точно по центру элементов**
   * Так система лучше распознает кнопки
   * Не кликайте на края элементов

3. **Используйте стандартные пути**
   * Если есть несколько способов - выберите самый простой
   * Пример: "Файл -> Создать" лучше, чем "Ctrl+N"

4. **Один шаг - одно действие**
   * Не пытайтесь записать сразу много действий
   * Лучше больше простых шагов, чем мало сложных

#### 4. Редактирование текстов

**Делайте инструкции понятными:**

Плохо:
```
"Кликните на кнопку"
```

Хорошо:
```
"Нажмите кнопку 'Создать' в левом верхнем углу экрана"
```

Ещё лучше:
```
"Нажмите зелёную кнопку 'Создать' в левом верхнем углу, 
чтобы создать новый документ из шаблона"
```

**Используйте:**
* Подробные описания
* Указание цели действия ("чтобы...")
* Уточнение местоположения ("в левом углу")
* Описание визуальных особенностей ("зелёная кнопка")

### Специальные ситуации

#### Работа с формами

**Если нужно заполнить несколько полей:**

1. Запишите клик на каждое поле отдельно
2. В инструкции укажите, что вводить:
   ```
   "Введите название документа в поле 'Имя файла'.
   Пример: 'Отчёт_2024'"
   ```

#### Работа с выпадающими списками

1. Кликните на выпадающий список
2. Кликните на нужный пункт
3. В инструкции укажите конкретное значение:
   ```
   "Выберите 'Microsoft Word' из списка программ"
   ```

#### Работа с диалоговыми окнами

**Проблема:** Окно может открыться в другом месте

**Решение:**
1. Не полагайтесь на точное положение окна
2. Описывайте элементы по их названиям:
   ```
   Хорошо: "Нажмите кнопку 'ОК' в появившемся диалоговом окне"
   Плохо: "Кликните в правом нижнем углу"
   ```

#### Ожидание загрузки

**Если действие требует времени:**

1. Поставьте запись на паузу
2. Дождитесь завершения загрузки
3. Продолжите запись
4. В инструкции добавьте:
   ```
   "Дождитесь, пока документ полностью загрузится
   (индикатор загрузки исчезнет)"
   ```

### Оптимальная длина гайда

**Рекомендации:**

* **Короткие гайды (3-7 шагов)**: Для простых задач
  * Пример: "Как сохранить документ"
  
* **Средние гайды (8-15 шагов)**: Основная часть гайдов
  * Пример: "Создание и отправка письма в Outlook"
  
* **Длинные гайды (16-30 шагов)**: Использовать осторожно
  * Пример: "Создание сложного отчёта в 1С"
  
* **Очень длинные (более 30 шагов)**: Разбить на несколько гайдов
  * Лучше сделать: "Часть 1: Подготовка", "Часть 2: Заполнение"

## Решение проблем

### Красная рамка не появляется

**Причины и решения:**

1. **Запись не запущена**
   * Нажмите "Start Recording" на панели управления

2. **Наведён курсор на фон**
   * Наведите курсор на кнопку/поле

3. **Приложение не поддерживается**
   * Проверьте, работает ли с другими приложениями (Word, Explorer)
   * Некоторые приложения не поддерживают UI Automation

### Шаг записался неправильно

**Что делать:**

1. НЕ останавливайте запись
2. Поставьте на **паузу** (Pause)
3. Продумайте, как записать правильно
4. Продолжите запись (Start Recording)
5. Запишите правильный шаг
6. После завершения - **удалите** неправильный шаг в редакторе

### Скриншот элемента не сохранился

**Причины:**
* Слишком быстро кликнули следующий элемент
* Система не успела сделать скриншот

**Как избежать:**
* Делайте паузу 2-3 секунды между кликами
* Не торопитесь при записи

### Сообщение "No screenshot available for this step"

Это нормально, если:
* Шаг - это системное действие (System Action)
* Приложение закрыло окно слишком быстро

Можно продолжать использовать гайд - скриншот не обязателен.

### Не могу сохранить гайд

**Проверьте:**

1. Указали название гайда
2. Указали автора
3. У вас есть права на запись в папку Documents
4. На диске достаточно места

### Редактор шагов не открывается

**Решение:**

1. Закройте FlowGuide Recorder
2. Откройте снова от имени администратора
3. Попробуйте **"Редактировать существующий гайд"**

### Приложение зависло

**Действия:**

1. Откройте Диспетчер задач (Ctrl+Shift+Esc)
2. Найдите "FlowGuide.Recorder.exe"
3. Нажмите "Снять задачу"
4. Перезапустите приложение

> [Внимание]: Несохранённые гайды будут потеряны!

## Примеры готовых гайдов

### Пример 1: Простой гайд (5 шагов)

**"Как сохранить документ в Word"**

1. Нажмите вкладку "Файл" в левом верхнем углу
2. Выберите "Сохранить как" из меню
3. Нажмите "Обзор", чтобы выбрать место сохранения
4. Введите название документа в поле "Имя файла"
5. Нажмите кнопку "Сохранить"

### Пример 2: Средний гайд (12 шагов)

**"Создание события в Календаре Windows"**

1. Нажмите Win (откроется меню Пуск) - System Action
2. Найдите и кликните на приложение "Календарь"
3. Нажмите кнопку "Создать" в левом верхнем углу
4. Введите название события в поле "Название"
5. Кликните на поле "Дата начала"
6. Выберите нужную дату в календаре
7. Кликните на поле "Время начала"
8. Установите время начала события
9. Кликните на поле "Дата окончания"
10. Выберите дату окончания
11. Добавьте описание в поле "Дополнительные сведения"
12. Нажмите кнопку "Сохранить"

### Пример 3: Системные действия

**"Открытие настроек дисплея в Windows"**

1. Нажмите Win (откроется меню Пуск) - System Action: OpenStartMenu
2. Откройте Параметры Windows - System Action: OpenSettings, ms-settings:display
3. (Автоматически откроется страница настроек дисплея)

## Чек-лист создания качественного гайда

Перед сохранением гайда убедитесь:

### Подготовка
- [ ] План действий составлен
- [ ] Рабочее пространство подготовлено
- [ ] Тестовые данные готовы
- [ ] Уведомления отключены

### Запись
- [ ] Между кликами паузы 2-3 секунды
- [ ] Нет лишних кликов
- [ ] Все важные шаги записаны
- [ ] Последовательность действий логична

### Редактирование
- [ ] Все инструкции понятны и подробны
- [ ] Указаны цели действий ("чтобы...")
- [ ] Нет грамматических ошибок
- [ ] Удалены лишние шаги
- [ ] Правильно настроены System Actions (если есть)

### Проверка
- [ ] Название гайда описательное
- [ ] Описание гайда заполнено
- [ ] Автор указан
- [ ] Гайд сохранён

### Тестирование
- [ ] Запустили гайд в Player
- [ ] Все шаги работают корректно
- [ ] Инструкции понятны другим пользователям
