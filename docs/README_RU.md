## 1. Обзор

FlowGuide — это служебная программа для автоматизации действий и обучения на платформе Windows 10/11. Она позволяет разработчикам, техническим писателям и специалистам службы поддержки записывать интерактивные пошаговые инструкции (гайды) внутри любого стандартного Windows-приложения. Записанные руководства могут воспроизводиться в виде оверлея поверх экрана пользователя, ведя его по этапам рабочего процесса и подсвечивая целевые кнопки, поля ввода и элементы управления в режиме реального времени.

Благодаря использованию встроенного Windows API UI Automation и глобальных перехватчиков (хуков) мыши, FlowGuide обеспечивает плавный процесс адаптации и обучения непосредственно внутри целевой программы, избавляя от необходимости обращаться к статичным внешним руководствам или видеозаписям.

## 2. Ключевые возможности

* **Запись действий**: Автоматическое отслеживание событий клика с использованием низкоуровневого глобального хука мыши Win32.
* **Умный выбор UI-элементов**: Проверка элементов под курсором с помощью UI Automation и сбор ключевых параметров (AutomationId, Name, ClassName и иерархия предков).
* **Автоматическое создание скриншотов**: Сохранение вырезанных областей со скриншотами подсвеченных элементов с небольшой задержкой после клика для стабилизации изображения.
* **Интерактивный оверлей подсветки (эффект прожектора)**: Затемнение экрана во время воспроизведения с созданием прозрачной «дыры» вокруг активного элемента управления.
* **Защита кликов (Click Guard)**: Блокировка нажатий пользователя за пределами активной подсвеченной зоны, обеспечивающая строгое соблюдение шагов инструкции.
* **Автовыполнение системных действий**: Автоматический запуск стандартных команд Windows (таких как открытие меню «Пуск», страниц параметров, командной строки или проводника) от имени пользователя.
* **Независимость от DPI и разрешения**: Запись относительных координат (нормализованных относительно размеров экрана и активного окна) для корректного масштабирования оверлея при любых настройках масштабирования интерфейса.
* **Редактирование шагов**: Просмотр, переопределение, удаление и уточнение записанных шагов, текстов инструкций и параметров выполнения.

| Слой / Компонент | Технология | Описание / Назначение |

| Платформа | .NET 8.0 | Целевой фреймворк: `net8.0-windows` |

## 4. Начало работы

### Требования

* Операционная система Windows 10 или Windows 11 (x64)
* Установленный .NET 8.0 SDK или .NET 8.0 Desktop Runtime

### Сборка проекта

Клонируйте репозиторий и скомпилируйте решение с помощью интерфейса командной строки .NET CLI:

```powershell
dotnet restore
dotnet build FlowGuide.Desktop.sln -c Release
```

### Запуск приложений

#### FlowGuide Recorder (Запись)

Утилита записи фиксирует действия пользователя. Обратите внимание, что глобальные хуки требуют прав администратора для перехвата событий окон других приложений, запущенных с повышенными привилегиями.

```powershell
dotnet run --project FlowGuide.Recorder/FlowGuide.Recorder.csproj
```

#### FlowGuide Player (Воспроизведение)

Интерактивный плеер отображает пошаговые инструкции поверх экрана.

```powershell
dotnet run --project FlowGuide.Player/FlowGuide.Player.csproj
```

## 6. Развертывание

Для сборки переносимых (portable) исполняемых файлов как для рекордера, так и для плеера выполните скрипт:

```powershell
powershell -ExecutionPolicy Bypass -File .\publish.ps1
```

После завершения компиляции в каталоге `.\Release` будут созданы одиночные исполняемые файлы, готовые к работе без предварительной установки.
